Naming Conventions:
- assets filenames: kebab-case
- JS filenames: Pascal Case
- variable & function names: camelCase

Directory:
- assets (for images, fonts, and other resources)
- src
  - component (for custom React Native components, styles within the same JS file)
  - screens (screens/pages for the React Navigation)
    - rootScreens (root screens part of RootStackScreens)
  - Styles.js (JS file for global styles definition)

Icons: Octicons

Dependencies Packages:
- React Navigation
- React Native Elements
- react-native-picker
- react-native-modal-datetime-picker
- react-native-linear-gradient
- react-native-svg & react-native-svg-transformer (dev dep.)

React Navigation:
https://reactnavigation.org/docs/getting-started
	yarn add @react-navigation/native
	yarn add react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view
	yarn add @react-navigation/stack

React Native Elements:
https://reactnativeelements.com/docs/
	yarn add react-native-elements
	yarn add react-native-vector-icons
	yarn add react-native-safe-area-context

react-native-picker/picker:
https://github.com/react-native-picker/picker

https://github.com/mmazzarolo/react-native-modal-datetime-picker
