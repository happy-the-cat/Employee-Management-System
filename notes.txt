Naming Conventions:
- assets filenames: kebab-case
- JS filenames: Pascal Case
- variable & function names: camelCase

Directory:
- assets (for images, fonts, and other resources)
- src
  - component (for custom React Native components, styles within the same JS file)
  - screens (screens/pages for the React Navigation)
    - rootScreens (root screens part of RootStackScreens)
  - Styles.js (JS file for global styles definition)

Icons: Octicons

Dependencies Packages:
- React Navigation
- React Native Elements
- react-native-picker
- react-native-modal-datetime-picker
- react-native-linear-gradient
- react-native-svg & react-native-svg-transformer (dev dep.)
- React Native Encrypted Storage
- react-native-get-random-values & NanoID (no need install)
- React Native Floating Action
- react-native-animatable
